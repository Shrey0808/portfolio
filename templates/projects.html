Road Lane Detection for Autonomous Vehicles

AIM: To develop a system that detects lane lines in real-time from video inputs to aid autonomous driving.

  • Used computer vision techniques to take video input from the cameras of an autonomous driving vehicle and returned a video with the road lanes marked for the vehicle to decide its path.
  • Implemented various computer vision techniques like color masks, grayscaling, Gaussian smoothing, Canny edge detection, region selection, and Hough transformation.
  • Took video input and divided it into multiple frames, applied the various CV Techniques, and combined all the frames using MoviePy.
  • Technologies: Python, OpenCV, MoviePy

AI Job Finder

AIM: To provide AI-based job recommendations using ML algorithms and real-time job data scraping from multiple websites.

  • The model scrapes 200 job listings in real-time from LinkedIn, Internshala, Timesjob, and Naukri.com, fetching the top 40 jobs from each site based on the user's inputs for job title, experience, and location.
  • User inputs are collected through a user-friendly interface, with Django powering the backend and HTML/CSS/JavaScript creating a responsive frontend. The fetched job results are then displayed on the website, allowing users to either view all jobs at once or filter them based on the platform.
  • The standout feature of the model is its AI recommendation system. It converts user inputs into vectors and employs cosine similarity to rank and provide the top job recommendations from the fetched listings.
  • Technologies: Python, Django, MongoDB, HTML, CSS, JavaScript, Bootstrap, Selenium, Beautiful Soup, TensorFlow, Scikit-learn

Handwritten Number Predictor

AIM: To predict handwritten numbers from image inputs using deep learning and computer vision techniques.

  • The first base model is a Digit Recognition System trained using the MNIST dataset with 60,000 training and 10,000 testing images, i.e., 6000 and 1000 images respectively for each unique digit from 0-9.
  • Employed Convolutional Neural Networks (CNN) to train the model, achieving an accuracy of 99.10% on the testing dataset.
  • The second advanced model uses computer vision techniques to segment the image of a handwritten number into individual digits and make predictions for each digit.
  • The frontend was designed and implemented using HTML, CSS, and JavaScript, and the backend was deployed using Flask.
  • Technologies: Python, TensorFlow, Scikit-learn, Computer Vision (CV)
